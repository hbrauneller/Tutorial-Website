JAVASCRIPT CONCEPTS



FUNCTIONS

Functions allow us to execute some action or actions and they are able to be used in a repeated fashion. There are four main characteristics of a function:

  1. They take some input given to them.
  2. They process the input given to them.
  3. They usually, but not always, return some value.
  4. They can be invoked as many times as we want, cutting down on code repition.


OBJECTS

Objects are a data type that let us store a collection of properties and methods. They begin with naming a variable, followed by a set of curly braces. Within the braces are:

  • KEYS: Identifying properties of the object, such as a name or a color. These are followed by a colon (:).
  • VALUES: The value describes the object, in answer to a particular key. Each value is followed by a comma (,).



ARRAYS

An array is a numerically indexed map of values. Starting out, it's easier to just say it's a great way to collect a bunch of values together. They allow us to collect things like names, usernames, product names, prices, etc. They can also be used to list data types in an ordered or numbered way.



FOR LOOPS

Loops are a method that allows you to do something repeatedly, even infinitely. A For Loop is a basic loop that can use many variables. Loops have a basic structure:

  1. Creation of an indexing variable
  2. A run condition
  3. Change to the indexing variable


CONDITIONALS

Conditionals are used to check certain conditions in your application. Conditionals have the power to alter the state of your application based on certain conditions being met or unmet, true or untrue.
